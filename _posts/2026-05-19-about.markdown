`HAVNO` is a pixel art adventure game based inspired by `DANIEL DEFOE`'s Robinson Crusoe, first published in 1719.
The game is splitted into several chapter, depending on development progress.
You can now enjoy the Full Chapter #1, <i>Back to the Shore</i>. If you finish all the enigmas,a teaser image of Chapter #1, <i>The Mist, will congratulate all your efforts.

An example of running a, instance of the Game locally using [python3] HTTP Webserver :
<ul>
<li>First, copy the assets to a chosen directory (i.e. MYPATH\MYGAME).</li>
<li>Install Python.</li>
<li>Run Powershell or Terminal on Linux/BSD systems.</li>
<li>Then, type the following:</li>
</ul>
{% highlight ruby %}
cmd
 python3 -m http.server
#=> Or in Python 2.x
 python -m http.server
{% endhighlight %}
