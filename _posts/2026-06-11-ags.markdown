`HAVNO` has been developped using Chris Jones [Adventure Game Studio] editor. `AGS` allows to create Sierra and LucasArts-style graphical adventure games. It is based on a `ROOM` system, each room being connected to an another and storing Walkable areas, Objects and various interactions. `AGS` is a simple way to create and compile your own adventure game.
